## Image Guidelines

### Format
- **Preferred**: JPG for photos, PNG for diagrams/charts with transparency
- **Size**: Optimize for web (under 500KB each)
- **Dimensions**: 800-1200px width (maintain aspect ratio)

### Quality Standards
- High resolution (at least 72 DPI for screen)
- Clear text (if text is included in image)
- Appropriate colors (not overly bright or distracting)
- Professional appearance

### Sourcing
- **Original Creation**: Best option - create your own diagrams
- **Public Domain**: Use historical photos/documents in public domain
- **Creative Commons**: Use CC0 or CC-BY licensed images (give attribution)
- **Fair Use**: Educational use of copyrighted material (limited portions)

### Attribution
When using third-party images, add attribution below the image:
```markdown
![Image description](images/filename.jpg)
*Source: [Attribution text] - Used under [license type]*
```

## Adding Images to Study Guides

### Markdown Syntax
```markdown
![Alt text](images/filename.jpg)
```

### Centered with Caption
```markdown
<p align="center">
  <img src="images/filename.jpg" alt="Description" width="700">
  <br>
  <em>Figure 1: Caption text here</em>
</p>
```

### Multiple Images Side-by-Side
```html
<p align="center">
  <img src="images/image1.jpg" width="45%" alt="Description 1">
  <img src="images/image2.jpg" width="45%" alt="Description 2">
  <br>
  <em>Comparison of two concepts</em>
</p>
```
